# AliasVault 0.9.0: First Public Beta Release

We're thrilled to announce the first public beta release of AliasVault 0.9.0, introducing new administrative features and system improvements.

## New Administrative Features

Enhanced management capabilities:

- New statistics dashboard for monitoring system usage
- Server settings page for easy configuration
- Background service for automatic log cleanup
- Improved system maintenance capabilities

## System Improvements

Infrastructure and deployment enhancements:

- Custom port settings moved to .env file for better configuration management
- Automatic download of latest Docker Compose files during installation and updates
- New TaskRunner service for scheduled background tasks
- Optimized system resource management

## Technical Details

Under the hood improvements include:

- Enhanced logging system with automatic cleanup
- Improved system monitoring capabilities
- Streamlined deployment process
- Better configuration management
