# Improved Password Generator and User Experience

We're excited to announce the release of AliasVault 0.15.0, packed with new features and improvements that enhance both security and user experience.
This release makes the password generation settings customizable, improves existing functionality, and adds better analytics.

## Customizable Password Generation

The latest version introduces powerful password generation customization:

- Configure password patterns globally or per credential
- Adjust length, character types, and complexity
- Control use of lowercase, uppercase, special, and ambiguous characters
- Browser extension now respects these settings for new aliases

## Improved Browser Extension

Enhanced autofill capabilities make your browsing experience smoother:

- Better compatibility with websites that previously rejected autofilled passwords
- More reliable form filling across different platforms
- Improved user experience for credential management

## Enhanced Admin Dashboard

New analytics features provide better insights:

- Comprehensive analytics charts for better data visualization
- Tracking of returning users
- Improved admin listings and search functionality
- Better management of email claims and disabled aliases

## Technical Improvements

Under the hood improvements include:

- Simplified installation process
- Windows WSL compatibility fixes
- Updated dependencies for better performance and security
- Enhanced server-side email claim management
