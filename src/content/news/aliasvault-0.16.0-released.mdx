# Enhanced Import Capabilities and Streamlined UI

We're excited to announce the release of AliasVault 0.16.0, featuring a powerful new import wizard and significant UI improvements that make managing your credentials even easier.

  src="/images/blog/releases/0.16.0.png"

## New Import Wizard

The latest version introduces a comprehensive import wizard that makes migrating to AliasVault seamless:

- Import credentials from popular password managers including 1Password, Bitwarden, Chrome, Firefox, KeePass, KeePassXC, and Strongbox
- Bulk favicon extraction for imported items
- Duplicate detection to prevent importing the same items multiple times
- User-friendly interface for a smooth migration experience

## Streamlined UI for Traditional Credentials

We've enhanced the user interface to better support traditional username/password credentials:

- Updated main app and browser extensions to prioritize relevant information
- Improved support for credentials with just username and password
- More intuitive display of legacy credentials
- Better user experience for both random aliases and traditional credentials

## Quality-of-Life Improvements

Numerous enhancements to make your experience smoother:

- Create custom credentials directly from the browser extension
- Improved autofill functionality with better form detection
- Enhanced vault search capabilities
- Better admin user management
- Non-blocking loading indicators that match AliasVault's design
- Improved browser extension form detection
- Better error messages for browser extension login
- Language-aware identity generator in browser extension

## Technical Improvements

Under the hood improvements include:

- Optimized display of legacy credentials
- Fixed SVG favicon rendering in browser extension
- Updated dependencies for better performance and security
- Various bug fixes and optimizations
