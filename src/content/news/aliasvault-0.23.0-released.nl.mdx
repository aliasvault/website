AliasVault is een open-source, privacy-first wachtwoordmanager met een ingebouwde e-mail server en alias generator. Volledig van Nederlandse bodem.

De nieuwste versie, 0.23.0, introduceert een nieuwe **all-in-one Docker image** voor eenvoudigere self-hosting, voegt **mobiele app verbeteringen** toe inclusief CSV export, en voegt overige verbeterde functies toe op alle platforms. Ook is het officiele GitHub project verhuisd naar de nieuwe `aliasvault` organisatie.

## üê≥ 1. All-in-One Docker image nu officieel beschikbaar

Op verzoek van de community biedt AliasVault nu een **all-in-one Docker image** die self-hosting aanzienlijk eenvoudiger maakt. Deze nieuwe image is perfect voor:

- **Eenvoudige Docker/Docker Compose setups**
- **NAS-apparaten** zoals QNAP of Synology
- **Snelle tests en ontwikkelomgevingen**

De all-in-one image bevat alle benodigde componenten in een enkele container, waardoor het veel toegankelijker wordt voor gebruikers die AliasVault willen draaien zonder meerdere Docker containers te beheren. Bijgewerkte installatie- en migratie-instructies zijn nu beschikbaar in onze [documentatie](https://docs.aliasvault.net/installation).

## üì± 3. Verbeterde mobiele app functies

### Configureerbare wachtwoord generator
Wachtwoord generator instellingen (lengte, complexiteit, etc.) zijn nu **configureerbaar direct vanuit de mobiele app**. Geen noodzaak meer om eerst in te loggen op de web-app - je kunt je wachtwoord generatie voorkeuren direct aanpassen vanaf je telefoon.

### Offline CSV export
Mobiele apps kunnen nu **je kluis exporteren naar CSV**, zelfs in **offline modus**. Dit betekent dat als je server niet beschikbaar is, je nog steeds toegang hebt en je credentials veilig kunt back-uppen. Dit is vooral handig voor:

- **Backup doeleinden** tijdens reizen
- **Data migratie** tussen instanties
- **Noodtoegang** wanneer de server niet bereikbaar is

### Verbeterde trackpad input
Verbeterde touch en trackpad input zorgt voor betere responsiviteit op alle mobiele apparaten.
