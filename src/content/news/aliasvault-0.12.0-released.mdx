# Chrome Extension

We're excited to announce the release of AliasVault 0.12.0, introducing the Chrome browser extension and enhanced account management features.
This release marks the first step in making AliasVault available on more systems with native clients, starting with the browser environment.

## Chrome Browser Extension

New browser integration for Google Chrome:

- Available on the [Chrome Web Store](https://chromewebstore.google.com/detail/aliasvault/bmoggiinmnodjphdjnmpcnlleamkfedj)
- Easy credential creation and autofill
- Seamless integration with login and registration forms
- Manual installation option for advanced users

## Enhanced Account Management

New features for better account control:

- Self-service account deletion option
- Improved security settings
- Optional IP address logging
- Better user privacy controls

## User Interface Improvements

Enhanced visual experience:

- New table view for credentials
- Improved default sort order
- Better mobile responsiveness
- Enhanced dark mode support

## Technical Improvements

Under the hood improvements include:

- Updated dependencies for better performance
- Enhanced security measures
- Improved error handling
- Better API stability
