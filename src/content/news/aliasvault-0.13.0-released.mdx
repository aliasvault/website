# AliasVault 0.13.0: Firefox and Edge Support

We're excited to announce the release of AliasVault 0.13.0. This release makes the AliasVault browser extension avaialble for the Firefox and Microsoft Edge browsers.

## New Browser Support

AliasVault is now available on two additional browser platforms:

- Firefox extension available at [Mozilla Add-ons](https://addons.mozilla.org/en-US/firefox/addon/aliasvault/)
- Microsoft Edge extension available at [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/aliasvault/kabaanafahnjkfkplbnllebdmppdemfo)
- Manual installation options for both browsers
- Full API compatibility with new browser clients

## Enhanced Browser Extension

Improved functionality across all supported browsers:

- Better form injection and testability
- Improved user registration and tutorial flow
- Version information displayed in settings
- Enhanced favicon handling
- More robust autofill behavior

## Admin Dashboard Improvements

New features for better administration:

- Additional statistics and metrics
- Improved troubleshooting capabilities
- Enhanced user management tools
- Better vault revision management

## Technical Improvements

Under the hood improvements include:

- Optimized vault storage
- Enhanced form detection
- Updated dependencies for better performance
- Improved error handling and logging
