# AliasVault 0.11.0: Email Attachments and Enhanced Security

We're excited to announce the release of AliasVault 0.11.0, introducing email attachment support and various security improvements.

## Email Attachment Support

New capabilities for handling email content:

- Support for receiving and opening email attachments
- File size limit of 10 MB per attachment
- Secure storage and retrieval of attachments
- Improved email viewing experience

## Security Enhancements

Improved protection and stability:

- Prevention of email address collisions
- Enhanced identity generation
- Better error handling and logging
- Improved security measures

## Installation Improvements

Simplified setup process:

- Streamlined installation procedure
- Better configuration options
- Enhanced error reporting
- Improved documentation

## Technical Improvements

Under the hood improvements include:

- Updated dependencies for better performance
- Enhanced error handling
- Improved test coverage
- Better code quality
