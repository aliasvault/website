## New Features

We managed to add a lot of improvements and new features to AliasVault since the last 1.000 GitHub Star Update:

### Multi-Language Support
- Added comprehensive multi-language support across web app, browser extensions, and mobile apps
- Over 20 community members contributing translations via [Crowdin.com](https://crowdin.com/project/aliasvault)
- Now supporting Chinese, Finnish, French, German, Hebrew, Italian, Portuguese (Brazilian), Russian, Turkish, Ukrainian and more

### Enhanced Security Features
- **Automatic clipboard clearing** across all platforms after copying sensitive values
- **Clickjacking mitigations** with advanced protection against malicious attacks

### Self-Hosting Improvements
- **All-in-one Docker container** for easier local deployment and NAS setups
- Simplified configuration and migration between standard and all-in-one setups

### Import Capabilities
- **Expanded import options** from existing password managers including Dropbox Passwords
- **KeePass CSV import** with improved special character handling

### UI/UX Enhancements
- **Offline CSV export** capability in mobile apps
- **Auto-lock timeout options** in browser extension
- **Password visibility toggles** on login screens across all platforms
- **Email domain selectors** for easier alias management
- Countless other improvements based on community feedback

## What's Coming: Passkey Support

We're excited to share that **Passkey support is coming to AliasVault**! This is a major step forward in making digital identity management even more secure and user-friendly.

### Current Progress

In our development branch, the browser extension has already implemented the base WebAuthn specification, which means:

- **Create Passkeys**: Generate secure passkeys directly in your browser
- **Secure Storage**: Store passkeys safely in your vault with end-to-end encryption
- **Seamless Login**: Use your passkeys to authenticate across supported services

### Timeline

In the next weeks we'll be continuing work focusing on:

- **Browser extension finalization**: Completing the remaining Passkey browser extension features
- **Mobile app integration**: Adding Passkey support to mobile applications
- **Cross-platform compatibility**: Ensuring seamless experience across all devices

This represents a significant leap forward in passwordless authentication, making AliasVault even more powerful for managing your digital identity securely.
